Group Sections

Blocks that share the same column alignment get clustered into sections. Four empty rows between groups? That's a pod boundary. A grid label appears? New section starts. The 20-rack pod heuristic (2 serpentine rows of 10) tags standard pods automatically.

Assign Hierarchy

Sections get placed into halls. Three strategies, tried in order: match by hall header column overlap, infer from spatial column gaps (8+ columns apart = separate hall), or fall back to a single layout group. The result is a full tree: site → halls → grids → pods → racks.
